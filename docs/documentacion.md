# Documentación del Proyecto DAW

## Descripción
Proyecto web básico funcional desarrollado como práctica de Git y GitHub.

## Estructura del proyecto
- `index.html` → página principal
- `css/styles.css` → estilos
- `js/app.js` → funcionalidad JavaScript
- `php/conexion.php` → conexión a base de datos
- `docs/` → documentación
- `README.md` → información general

## Ramas
- `main` → producción
- `desarrollo` → desarrollo activo

## Flujo de trabajo
1. Crear rama `desarrollo`
2. Realizar cambios y commits descriptivos
3. Merge a `main` cuando todo funcione
